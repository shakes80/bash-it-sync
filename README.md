# bash-it-sync
by:shakes80

made to install and sync `https://github.com/Bash-it/bash-it`

to install run: 
```
git clone --depth=1 https://github.com/shakes80/bash-it-sync.git ~/bash-it-sync
cd ~/bash-it-sync
./install.sh
```

Optional: for more secure logout run
```
./install_secure_logout.sh
```
